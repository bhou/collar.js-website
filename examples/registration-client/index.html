collar.js example : registration demo


The register button will be enabled, when you enter a non-registered email, and a valid password (Minimum 8 characters at least 1 Alphabet and 1 Number).

The validation model of the view: 


{
  emailStatus :  "ok", // "ok" or error message
  passwordStatus : "ok", // "ok" or error message
}
By using collar.js' [model] node, you can easily bind your model to your view.
